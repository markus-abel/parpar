parpar by markus-abel

particles in turbulent flows - parallelized MPI version

We aim at a framework for particle integration with velocity given externally by a CFD code. The project will evolve in several stages. First we program an independent particle evolution, then coordination with the underlying cfd code is done by manual load balancing. Eventually we might arrive at automatic load balancing.
